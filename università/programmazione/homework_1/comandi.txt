Comandi per la compilazione separata

//Nel terminale, nella directory della cartella contenente i file

//Compilazione dei singoli file in file oggetto
gcc -c magazzino.c
gcc -c main.c

//Assemblaggio dei file sorgente nell'unico file programma di output
gcc main.o magazzino.o -o programma

//Per runnare (su mac)
./programma
